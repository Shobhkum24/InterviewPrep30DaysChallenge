## How to Use

1. Follow one day at a time — don't skip ahead
2. Solve the DSA problem **before** reading the solution
3. Time yourself (20-45 min per problem)
4. Draw the system design on paper/whiteboard first
5. Record mock interviews in week 5
6. Check off items in the daily checklist
7. Write notes — what you learned, what to review
